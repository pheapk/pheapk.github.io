# 2D Breakout Game

- Pure JavaScript
- HTML 5 canvas
  
  ## To Play The Game
  - Download the index.html
  - Load it on browser
  - Control the game using left / right arrow
  - Or use mouse
  - You have 3 lives
  
  ## Future version
  - More levels
  - Different brick layouts and properties
  
  Source: from tutorial at https://developer.mozilla.org
